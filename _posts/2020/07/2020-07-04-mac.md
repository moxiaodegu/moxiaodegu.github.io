## 安装node报错

Mac M1 中，nvm install 可能会安装不了，报clang: error: no such file or directory: 'CXX=c++'错误
clang: error: no such file or directory: 'CXX=c++'

解决方案：在iTrem2 中输入`arch -x86_64 zsh`
